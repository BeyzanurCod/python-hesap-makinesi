🧮 Python Hesap Makinesi
Bu proje, temel matematiksel işlemleri gerçekleştiren, işlem geçmişini kaydeden ve kullanıcı dostu bir komut satırı arayüzüne sahip olan bir Python uygulamasıdır.

✨ Özellikler
Temel İşlemler: Toplama, çıkarma, çarpma ve bölme işlemlerini saniyeler içinde yapar.

Hata Yönetimi: Geçersiz girişlerde (harf girilmesi, sıfıra bölme vb.) programın çökmesini engelleyen dinamik kontrol yapısı.

Geçmiş Kaydı: Yapılan tüm işlemler tarih ve saat bilgisiyle birlikte gecmis.txt dosyasına otomatik olarak kaydedilir.

Sürekli Kullanım: Kullanıcı çıkış yapana kadar döngü içinde çalışmaya devam eder.

🛠️ Kullanılan Teknolojiler
Dil: Python 3.x

Kütüphaneler: datetime (Zaman damgalı kayıtlar için), os
